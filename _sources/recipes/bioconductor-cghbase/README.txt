bioconductor-cghbase
====================

|downloads|

Contains functions and classes that are needed by arrayCGH packages.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/CGHbase.html
Versions 1.32.0
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-cghbase
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install bioconductor-cghbase

and update with::

   conda update bioconductor-cghbase

bioconductor-goseq
==================

|downloads|

Detects Gene Ontology and/or other user defined categories which are over/under represented in RNA-seq data

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/goseq.html
Versions 1.22.0
License  LGPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-goseq/1.22.0
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install bioconductor-goseq

and update with::

   conda update bioconductor-goseq

functools32

Backport of the functools module from Python 3.2.3 for use on 2.7 and PyPy.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install functools32

and update with::

   conda update functools32

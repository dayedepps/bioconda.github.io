bioconductor-camera
===================

|downloads|

Annotation of peaklists generated by xcms, rule based annotation of isotopes and adducts, EIC correlation based tagging of unknown adducts and fragments

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/CAMERA.html
Versions 1.26.0, 1.22.0
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-camera/1.26.0
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install bioconductor-camera

and update with::

   conda update bioconductor-camera

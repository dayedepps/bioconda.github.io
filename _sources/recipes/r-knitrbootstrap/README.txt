================

A framework to create Bootstrap 3 HTML reports from knitr Rmarkdown.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-knitrbootstrap

and update with::

   conda update r-knitrbootstrap

Mugsy is a multiple whole genome aligner.

Versions 1.2.3

Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/mugsy

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install mugsy

and update with::

   conda update mugsy

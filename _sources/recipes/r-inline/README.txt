.. _`r-inline`:

r-inline
========

|downloads|

Functionality to dynamically define R functions and S4 methods with inlined C, C++ or Fortran code supporting .C and .Call calling conventions.

======== ===========
Home     
Versions 0.3.14
License  LGPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-inline
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-inline

and update with::

   conda update r-inline

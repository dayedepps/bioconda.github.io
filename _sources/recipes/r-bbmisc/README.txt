r-bbmisc
========

Miscellaneous helper functions for and from B. Bischl and some other guys at TU Dortmund, mainly for package development.

Home     https://github.com/berndbischl/BBmisc
Versions 1.9
License  BSD_3_clause + file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-bbmisc

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-bbmisc

and update with::

   conda update r-bbmisc

bioconductor-summarizedexperiment
=================================

|downloads|

The SummarizedExperiment container contains one or more assays, each represented by a matrix-like object of numeric or other mode. The rows typically represent genomic ranges of interest and the columns represent samples.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/SummarizedExperiment.html
Versions 1.0.2
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-summarizedexperiment/1.0.2
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install bioconductor-summarizedexperiment

and update with::

   conda update bioconductor-summarizedexperiment

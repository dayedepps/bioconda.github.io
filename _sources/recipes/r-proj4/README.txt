.. _`r-proj4`:

r-proj4
=======

|downloads|

A simple interface to lat/long projection and datum transformation of the PROJ.4 cartographic projections library. It allows transformation of geographic coordinates from one projection and/or datum to another.

======== ===========
Home     http://www.rforge.net/proj4/
Versions 1.0_8
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-proj4
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-proj4

and update with::

   conda update r-proj4

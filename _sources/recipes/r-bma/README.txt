r-bma
=====

|downloads|

Package for Bayesian model averaging and variable selection for linear models, generalized linear models and survival models (cox regression).

======== ===========
Home     http://www.r-project.org, http://www.research.att.com/~volinsky/bma.html
Versions 3.18.6
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-bma
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-bma

and update with::

   conda update r-bma

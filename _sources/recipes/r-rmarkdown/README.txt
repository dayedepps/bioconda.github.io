r-rmarkdown
===========

|downloads|

Convert R Markdown documents into a variety of formats.

======== ===========
Home     http://rmarkdown.rstudio.com
Versions 0.9.5
License  GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-rmarkdown
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-rmarkdown

and update with::

   conda update r-rmarkdown

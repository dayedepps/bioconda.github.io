bioconductor-genelendatabase
============================

|downloads|

Length of mRNA transcripts for a number of genomes and gene ID formats, largely based on UCSC table browser

======== ===========
Home     http://bioconductor.org/packages/release/data/experiment/html/geneLenDataBase.html
Versions 1.6.0
License  LGPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-genelendatabase/1.6.0
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install bioconductor-genelendatabase

and update with::

   conda update bioconductor-genelendatabase

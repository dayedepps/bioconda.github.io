r-phonr
=======

|downloads|

Tools for phoneticians and phonologists, including functions for normalization and plotting of vowels.

======== ===========
Home     http://drammock.github.io/phonR/
Versions 1.0_3
License  GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-phonr
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-phonr

and update with::

   conda update r-phonr

.. _`bioconductor-seqlogo`:

bioconductor-seqlogo
====================

|downloads|

seqLogo takes the position weight matrix of a DNA sequence motif and plots the corresponding sequence logo as introduced by Schneider and Stephens (1990).

======== ===========
Home     https://www.bioconductor.org/packages/release/bioc/html/seqLogo.html
Versions 1.36.0
License  LGPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-seqlogo
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install bioconductor-seqlogo

and update with::

   conda update bioconductor-seqlogo

r-bigmemory
===========

|downloads|

Create, store, access, and manipulate massive matrices. Matrices are allocated to shared memory and may use memory-mapped files.  Packages 'biganalytics', 'bigtabulate', 'synchronicity', and 'bigalgebra' provide advanced functionality.

======== ===========
Home     http://www.bigmemory.org
Versions 4.5.19
License  LGPL-3 | Apache License 2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-bigmemory
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install r-bigmemory

and update with::

   conda update r-bigmemory

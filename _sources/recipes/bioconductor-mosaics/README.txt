bioconductor-mosaics
====================

|downloads|

This package provides functions for fitting MOSAiCS, a statistical framework to analyze one-sample or two-sample ChIP-seq data.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/mosaics.html
Versions 2.4.1
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-mosaics
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install bioconductor-mosaics

and update with::

   conda update bioconductor-mosaics
